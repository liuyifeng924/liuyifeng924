(Python)实现TextureMerger打包图集

本文最后更新于：7 天前

TextureMerger的用法详解，使用命令行实现TextureMerger打包图集。

本例子采用python2.7.16版本

TextureMerger的用法：
- Texture Merger 的命令行，注意路径必须是绝对路径，路径好像不支持中文
- 命令格式：-p [目录] […] -o [json输出路径] -e [文件筛选正则表达式]
- example: TextureMerger -p e:/01main -o e:/01main.json -e /.*.(png)
