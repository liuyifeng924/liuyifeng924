this指向问题

this指向

  • 普通函数
  • 箭头函数

普通函数

this永远指向的是最后调用它的对象，也就是看它执行的时候是谁调用的。

//例子1
//引用logger的printName方法
//调用的时候是window环境
class Logger {
  printName(name = 'there') {
    this.print(`Hello ${name}`);
  }

  print(text) {
    console.log(text);
  }
}

const logger = new Logger();
const { printName } = logger;
printName(); //报错print no undefined

稍作修改，把printName方法绑定在logger类中。

class Logger {
  constructor() {
    this.printName = this.printName.bind(this);
  }
  
  printName(name = 'there') {
    this.print(`Hello ${name}`);
  }

  print(text) {
    console.log(text);
  }
}

const logger = new Logger();
const { printName } = logger;
printName(); //hello there

箭头函数

this指向它的上下文环境。

//箭头函数
class Obj {
  constructor() {
      this.name = "Lily";
      this.getName = () => this.name;
  }
}

const myObj = new Obj();
const { getName } = myObj ;
getName(); //Lily
