几个非常用运算符

二进制转化

以下运算符涉及二进制转化，此处说明一下十进制转换为二进制的方法：除二取余法。（除以2取余数，余数逆序排列.。二进制数一般为32位，不够的往前面添0。）

//十进制数转换为二进制
//例子一：5
//5/2 = 2 ... 1
//2/2 = 1 ... 0
//1/2 = 0 ... 1
//因此，（十进制数）5转化为（二进制数）就是101，二进制数一般为32位，不够的往前面添0。
//即0000 0000 0000 0000  0000 0000 0000 0101

//例子二：24
//24/2 = 12 ... 0
//12/2 = 6  ... 0
//6/2 = 3   ... 0
//3/2 = 1   ... 1
//1/2 = 0   ... 1
//因此，（十进制数）24转化为（二进制数）就是11000，二进制数一般为32位，不够的往前面添0。
//即0000 0000 0000 0000  0000 0000 0001 1000

//例子三：20
//20/2 = 10 ... 0
//10/2 = 5  ... 0
//5/2 = 2   ... 1
//2/2 = 1   ... 0
//1/2 = 0   ... 1
//即0000 0000 0000 0000  0000 0000 0001 0100

“<<”运算符

//"<<"运算符：用于对二进制数左移多少位。
1 << 0 //结果：1
1 << 1 //结果：2
1 << 2 //结果：4
1 << 10 //结果：1024
5 << 10 //结果：5120
//由上可得x << y 相当于 x * 2^y

//较为常用的例子：
const kb = 1 << 10; //1024
const mb = kb << 10; //1048576
const tb = mb << 10; 
const gb = tb << 10;

“&”运算符

//"&"运算符：用于对两个二进制操作数逐位“位与”进行比较。
//“位与”即1&&1==1, 1&&0==0, 0&&0==0
//平时我们使用的数字都是十进制数，该方法是对转化后的二进制数进行比较。
//"&"运算符就是比较二进制中为true的值提取出来，然后逐位计算true或false。

24 & 5 //结果：0
//具体规则：逐位进行比较，例如1 && 0 结果就是 0
//数值5： 0000 0000 0000 0000  0000 0000 0000 0101
//数值24：0000 0000 0000 0000  0000 0000 0001 1000
//结果：  0000 0000 0000 0000  0000 0000 0000 0000

24 & 20 //结果：16
//数值24：0000 0000 0000 0000  0000 0000 0001 1000
//数值20：0000 0000 0000 0000  0000 0000 0001 0100
//结果：  0000 0000 0000 0000  0000 0000 0001 0000

“^”运算符

//"^"运算符：用于对两个二进制操作数逐位“位异或”进行比较。
//“位异或”即1&&0==1 1&&1==0 0&&0==0

24 & 5 //结果：29
//具体规则：逐位进行比较，例如1 && 0 结果就是 1
//数值5： 0000 0000 0000 0000  0000 0000 0000 0101
//数值24：0000 0000 0000 0000  0000 0000 0001 1000
//结果：  0000 0000 0000 0000  0000 0000 0001 1101

“|”运算符

//"|"运算符用于对两个二进制操作数逐位进行“位或”比较。
//“位与”即1||1==1, 1||0==1, 0||0==0
//其他与"&"运算符相似

24 & 5 //结果：29
//具体规则：逐位进行比较，例如1 && 0 结果就是 1
//数值5： 0000 0000 0000 0000  0000 0000 0000 0101
//数值24：0000 0000 0000 0000  0000 0000 0001 1000
//结果：  0000 0000 0000 0000  0000 0000 0001 1101

“~”运算符

//"~"运算符：简单理解为取负运算，再减去1

~24 //结果：-25 即：-24-1
~10 //结果：-11 即：-10-1
